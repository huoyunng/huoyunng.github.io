什么是Hexo

Hexo是一款基于当下热门的Node.js开发出来的静态博客框架。用其配合Markdown语法编辑文章发布是非常快捷方便的。

为什么选择Hexo

因为Hexo静态博客框架已经把我们个人博客常用的功能都已经设计好了，不需要我们把过多经历花费在博客布局设计上。并且它是静态的，所以数据库设计，交互设计这些都不用了，加大了我们部署个人博客上线的效率和日常使用的便捷性。

Hexo的延展配件丰富，第三方插件兼容好，基于Hexo的博客样式主题同样丰富，使用用户多，版本更新迭代健康，还有一个重要点是官方配套文档写的很友善。

主要常用的命令：
hexo new “new a post”
hexo generate # 注意：每次创建新帖子，或更新内容，都需要执行 generate 重新生成新的静态文件
hexo server # server启动后，我们就可以要本地预览效果了
hexo deploy # 等本地预览没问题后，就可以部署到远端
hexo config # 很实用的配置命令，列表所有的hexo的配置信息，也就是_config.yml的文件内容,文档底部有config的介绍

提示: 可以用 hexo help查看hexo的常用命令
